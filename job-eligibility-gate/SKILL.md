# Job Eligibility Gate — 岗位资格四道闸

偏好数据源：`~/job-search-system/profile/career-preferences.yaml`（以其为准，本文为执行逻辑）。

## 第一闸：届别（最硬）

- 用户 = **2026.08 硕士应届毕业**（世宗大学）。
- 放行：明确「接受2026届 / 毕业不限 / 留学生专场 / 经验不限 / 0-1年 / 毕业2年内」。
- **2027届岗位不直接毙**：8月毕业是届别边界，部分公司把 2026.9 前后毕业划为 2027 届（赛轮/森麒麟/万华化学已核实可投）。规则：JD 未写死毕业时间区间的 2027 届岗 → 标「届别待核」，核实口径后放行或毙。
- 直接毙：明确「仅2027届且毕业时间2026.9之后」、要求3年以上经验。
- **应届身份影响标注**（每条推荐必带）：该岗若为正式入职需缴社保 → 标「⚠️影响央国企校招应届资格」；可实习入职/毕业前入职 → 标「✅可实习过渡保应届」。

## 第二闸：出海口径（2026-07-17 用户定死）

- 外企/国外公司：岗位对接中国业务或国际业务 → 放行（含外企在华 China marketing）。
- 中国公司：岗位必须明确对接海外客户/海外市场/跨境交易/海外收入 → 放行；纯国内业务 → 毙（S线韩企对韩业务除外——对韩=跨境）。
- 判断依据是岗位实际服务对象，不是公司国籍。报告必须写清服务市场。

## 第三闸：黑名单与去重

- 黑名单：拼多多/Temu 校招（届别必死，2026-07-16用户指令；已投的「商家运营·多语种」继续跟踪状态不受影响）。
- 去重底库：求职主表(Google Sheets 唯一版) + `~/job-search-system/state/*.csv` + Codex reports。同公司同岗位（含改名重发）不重复推荐；同公司默认只保留最匹配的1个岗。

## 第四闸：地点与薪资

- 地点：全国放开。优先级：长三角(杭/苏/宁/甬/沪)重点 > 青岛/烟台/山东 > 一线(非首选) > 其他；远程岗接受。
- 薪资：低于 8k 毙（仅过滤用，此数字禁止出现在任何投递材料中）。
- 快线标记：青岛/烟台韩企、对韩业务、可快速入职的岗位标「🚀快线」（用户8月起需要尽快上岗）。

## 输出

每岗一行：`公司 | 岗位 | 闸1✓/✗/待核 | 闸2✓/✗ | 闸3✓/✗ | 闸4✓/✗ | 应届影响 | 快线标记`。全✓才进评分环节（job-fit-and-tailor）。
